Start with a repeatable random sample from the normal distribution. Make it single-precision, because that's all the vMAT clustring functions can handle at present, and we want comparably precise results.

X = gallery('normaldata', [10 3], 13, 'single');

Compute the pairwise-distances between observations (pdist) and then the hierarchical cluster tree (linkage). The dendrogram plot provides a nice visual summary of the hierarchical cluster tree.

NOTE: Don't forget, vMAT_pdist wants X', and vMAT_linkage has zero-based indexes in the first two columns. The data in the MAT file includes the variable Zv for validating the test results.

Y = pdist(X);
Z = linkage(Y);
Zv = [Z(:,1:2)-1 Z(:,3)];
makeHtmlTable(Z,[], ...
    seqlabels('link', length(Z)), {'idx1', 'idx2', 'distance'}, cmap, 4);
dendrogram(Z);

Two methods of clustering are supported in vMAT: 'Cutoff' uses an inconsistency coefficient to produce an arbitrary number of clusters, and 'MaxClust' returns at most a specified number of clusters. [The MATLAB cluster function of course has a number of additional options, but these are the most commonly useful ones.]

The inconsistency coefficient is explained in the documentation for the inconsistent function.

W = inconsistent(Z);
ZW = [Z W];
makeHtmlTable(ZW, [], ...
    seqlabels('link', length(Z)), ...
    {'idx1', 'idx2', 'distance', 'mean', 'std', 'n', 'coefficient'}, cmap, 4);

Vmax4 = cluster(Z, 'MaxClust', 4);
